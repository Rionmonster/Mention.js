Remote Example: Name, Username, & Image

In this example, properties can be queried from a remote location (in this case JSON Placeholder) and will allow you to populate completion results from a URL (as opposed to a hard-coded array or collection)

Usage

To target a remote location to query, you simply need to add a remote property that points to the URL that you are targeting as seen below : 

$("#remote").mention({
    limit: 10,
    remote: 'http://jsonplaceholder.typicode.com/users' 
});
This will pass your current search term (the difference between the previous delimiter and the current cursor position) as a parameter called query to your remote location and it will return the elements that the query yields.
